# Handwritten Digit Generation using DCGAN

## Overview

This project uses a Deep Convolutional Generative Adversarial Network (DCGAN) to generate realistic handwritten digit images. The model is trained on the MNIST dataset and learns to create new digit images similar to real handwritten digits.

## Tech Stack

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
* DCGAN

## Features

* Deep Learning-based Image Generation
* GAN Training on MNIST Dataset
* Synthetic Handwritten Digit Creation
* Generator and Discriminator Networks
* Visualization of Generated Images
